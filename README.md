# hejnote. app

## Project setup
Add new file `.env.local`
```
VUE_APP_SUPABASE_URL=YOUR_SUPABASE_URL_WITH_HTTPS_IN_FRONT
VUE_APP_SUPABASE_KEY=YOUR_API_KEY
```
And then
```
npm install
```

### Compiles and hot-reloads for development
```
npm start
```

### Compiles and minifies for production
```
npm run build
```
